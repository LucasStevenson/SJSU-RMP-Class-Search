## How to run this application with Docker

1. Build

```sh
$ docker build . -t sjsu-rmp-class-search
```

2. Run

```sh
$ docker run --rm -p 5000:5000 -t sjsu-rmp-class-search
```

Open `http://localhost:5000` in your web browser
